> 이 문서는 comad-image · comad-pptx · comad-motion 세 스킬이 공유하는 프롬프트 패턴 지식 베이스다.

## 1. 핵심 원칙 (GPT-Image 2.0 시대)

### 1-1. 키워드 스태킹은 끝났다

과거 `masterpiece, ultra-detail, 8k, award-winning` 나열은 **효과 없음 → 오히려 간섭**. GPT-Image 2.0 은 자연어 해석 능력이 올라 **장문 서술이 오히려 정확**하다.

**증명된 비교**:
```
BAD:  "masterpiece, ultra-detail, 20대 아이돌, 일상 셀카, 따스한 분위기"
GOOD: "20대 초반 한국 여성 아이돌의 매우 사실적인 일상 셀카. 아이폰 전면 카메라로 촬영한 듯한 시점과 약간의 왜곡. 따뜻한 자연광, 피부에 부드럽게 닿는 햇빛. 캐주얼한 옷차림..."
```

### 1-2. 한국 시장 공통 어미

1500+ 한국 프롬프트를 분석한 결과 **반복되는 꼬리말 5 패턴**:
- `한국 시장의 인물 비주얼 문법에 맞는 현실적인 비율과 표정을 유지한다. AR 16:9.` (인물 80건+)
- `실제 라이프스타일 광고처럼 믿음직하게 연출한다. 등장 인물은 성인 한국인으로 보이게 하고` (광고)
- `재질감이 명확히 드러나도록 하고` (제품)
- `인터페이스와 재질감이 선명하게 읽히도록 하고` (UI/디지털)
- `모바일 친화적인 9:16 비율` (SNS/릴스)

### 1-3. 종횡비 분포 (실무 기준)

| 비율 | 용도 | 빈도 | 꼬리 표기 |
|---|---|---:|---|
| 16:9 | 히어로·키비주얼·Web | 143+ | `AR 16:9.` or `비율 16:9` |
| 4:5 | 이메일·CRM·Feed | 119+ | `AR 4:5.` |
| 9:16 | 릴스·쇼츠·스토리 | 74+ | `모바일 친화적인 9:16 비율` |
| 1:1 | Feed·썸네일 | 9 | `비율 1:1` |
| 21:9 | 울트라와이드·배너 | 3 | `21:9` |

## 2. OpenAI 공식 8-슬롯 구조

공식 문서 기준. 복잡한 시안에 적용.

```
1. 목적/용도           intended use
2. 핵심 브리프         core brief (one-line subject)
3. 필수 요소           essential elements (must-appear)
4. 맥락/환경           context/environment
5. 구도/공간 관계      composition + spatial relations
6. 빛/색/재질          light / color / material
7. 제약/금지/고정      constraints / prohibitions / fixed
8. 출력/포맷           output format + aspect ratio
```

### 최소 공식 (대부분 시안)
```
intended use + (background/scene → subject → key details → constraints)
```

예:
> "제주 주말 특가 패키지 전환용 여행 광고 이미지 [intended use],  
>  오션뷰 객실과 수영장이 한 화면에서 명확하게 보이는 분할 구성 [scene→subject],  
>  가격과 한정 특가 정보를 올릴 수 있는 명확한 여백 [key detail],  
>  한국 여행 플랫폼 광고 감성 [constraint/style],  
>  비율 4:5 [output]"

## 3. GPT 직접 입력 스타일 (Flat Natural Language)

**형태**: 쉼표로 구분된 긴 단일 문장. 487 샘플 평균 528자.

### 3-1. 템플릿
```
[용도 한 줄] + 이미지,
[핵심 장면/구도],
[분위기 키워드 2-3개],
[한국 감성/시장 맥락],
[여백·CTA·정보 영역 지시],
[비율 X:Y]
```

### 3-2. 실제 예 (P-004 · 호텔 키비주얼)
> 제주 부티크 호텔 가을 캠페인 키비주얼, 노을빛이 들어오는 프라이빗 테라스와 멀리 보이는 바다, 현무암 건축과 리넨 소재의 차분한 인테리어, 여유롭고 고급스러운 한국형 럭셔리 호스피탈리티 무드, 감성적이지만 지나치게 관광 포스터 같지 않은 상업 사진, 헤드라인을 배치할 넓은 여백, 기억에 남는 브랜딩 중심의 시즌 비주얼, **비율 16:9**

### 3-3. 실제 예 (P-001 · 릴스 썸네일)
> 2박 3일 서울 여행 코스 릴스 썸네일용 이미지, 루프탑 노을, 한옥 골목, 감성 카페, 호텔 침대 뷰, 편의점 야식 장면이 콜라주처럼 자연스럽게 연결된 구성, 한국 도시 여행 크리에이터 감성, 저장하고 공유하고 싶은 일정 추천 콘텐츠 무드, 활기 있지만 복잡하지 않은 구도, **모바일 친화적인 9:16 비율**

### 3-4. 체크리스트 (GPT 스타일)
- [ ] 첫 구문 = "~한 이미지" 로 용도 못박기
- [ ] 중심 오브제 + 보조 오브제 최소 3개 나열
- [ ] 한국적 맥락 1회 이상 삽입 ("한국", "한국형", "한국 시장의")
- [ ] 분위기 형용사 2-3개 (지나치지 않게)
- [ ] CTA·여백·정보 영역 지시 (커머스·광고일 때)
- [ ] 마지막 = **비율**

## 4. Codex /imagen 구조화 스타일 (11-Field Template)

**형태**: `키: 값\n` 으로 정렬된 구조화 블록. 1000 샘플 평균 600자.

### 4-1. 기본 11 필드 (범용)
```yaml
작업 방식: [팀 유형] 실무팀/스튜디오가 짧게 토론해 합의한 최종안처럼 생성한다.
실무 상황: [비즈니스 맥락 한 문장]

용도: [카테고리 — 아래 4-3 참조]
핵심 작업: [한 줄 제목]
장면 설명: [주 피사체 + 보조 + 스타일 톤]
구도: [시선 흐름 + 피사체-배경 관계]
연출 방향: [조명·무드·질감]
텍스트 처리: [카피 포함 여부 + 가독성]
와우 포인트: [한 문장 인상 + 차별화 포인트]
완성 기준: [체크 조건 3-5개]
제외할 것: [금지 요소 3-5개]
```

### 4-2. 용도별 추가 필드

**에디토리얼 포스터** (`용도: 에디토리얼 포스터`)
- 추가 없음. 기본 11 필드에 `강한 타이포 대비, 절제된 색상 수` 감성 추가.

**교육/인포그래픽** (`용도: 교육/인포그래픽`)
- `주제 설명` (장면 설명 대신)
- `정보 구조`: 섹션 구성 나열
- `도표 구성`: 그래프 타입 (막대/라인/지도/아이콘 박스)
- `세부 제어`: 범례·단위·한국어 표기 강제

**만화/코믹** (`용도: N컷 XXX 만화`)
- `컷 분할`: 대형 오프닝 1 + 긴장 2 + 결론 1 등
- `패널 리듬`: 큰→작은→큰 등 압축/확장 리듬
- `카메라`: 로우앵글/오버숄더/클로즈업/와이드
- `말풍선/캡션`: 컷당 1-2개, 한국어 자간 보호
- `의성어/효과`: 속도선·압박감, 마지막 컷 단순화

**캠페인 포스터** (`용도: 캠페인 포스터`)
- `전환 구조`: 히어로 → 핵심 포인트 → 브랜드 톤 → CTA 순서
- `행동 유도`: 저장/구매/예약 유도 구조
- `추가 고도화`: CTA 묻힘 방지, 과밀 방지

**광고 실사/키비주얼** (`용도: 광고 실사/키비주얼` or `실사 사진`)
- 연출 방향에 `영화적 조명, 공기감, 재질감` 필수
- `제외할 것`: 과한 HDR, 과포화, 플라스틱 피부

**브랜딩/디자인 레이아웃** (`용도: 브랜딩/디자인 레이아웃`)
- 구도에 `메인 오브제 우선` 명시
- `텍스트 처리`: 짧은 한국어 카피만

**디지털 작업환경 장면** (`용도: 디지털 작업환경 장면`)
- `연출 방향`: 실제 작업자 책상 밀도
- `완성 기준`: UI 텍스트·창 제목이 실제로 읽혀야
- `제외할 것`: 가짜 UI, 뭉개진 글자, 비현실 원근

## 5. "한국 감성" 키워드 사전

GPT-Image 2.0 이 한국 문법을 해석하는 트리거 구문. 프롬프트 안에 1개 이상 삽입하면 "한국스러운" 결과 확률 급상승.

| 맥락 | 사용 문구 |
|---|---|
| 인물 | `한국 시장의 인물 비주얼 문법에 맞는 현실적인 비율과 표정` |
| 인물 | `등장 인물은 성인 한국인으로 보이게 하고` |
| 인물 | `거울 없는 휴대폰 셀카처럼 보이게 하고` |
| 광고 | `한국 여행 플랫폼 광고 감성` |
| 광고 | `한국형 럭셔리 호스피탈리티 무드` |
| 광고 | `실제 라이프스타일 광고처럼 믿음직하게 연출` |
| 브랜드 | `한국 부티크 호텔 특유의 조용한 고급스러움` |
| 디자인 | `한국적 겨울 정취를 담은 미니멀 한지 레이어 아트` |
| 상업 | `한국 도시 여행 크리에이터 감성, 저장하고 공유하고 싶은` |
| UI | `한국어 자간이 무너지지 않게` |

## 6. 스타일 DNA (comad-motion/imagery.md 확장)

1500+ 샘플에서 관찰된 **6가지 주류 스타일 DNA**:

### DNA 1 · 에디토리얼 타이포 히어로
- 중앙 대형 타이포 + 기하 도형 + 2-4 절제된 색
- 용도: 포스터, 브랜드 선언, 컨퍼런스
- 핵심 구문: `대담한 에디토리얼 레이아웃, 극적인 여백, 전시 포스터급 그래픽 임팩트`

### DNA 2 · 시네마틱 매크로 사실성
- 단일 피사체 극close-up + 자연광/시네마 조명
- 용도: 식품·화장품·제품 상세
- 핵심 구문: `영화적 순간 포착, 재질감과 공기감, 현실보다 더 인상적인 사실성`

### DNA 3 · 정보 밀도 디지털 장면
- 실제 스크린·UI 복제 + 미세 텍스트 가독성
- 용도: SaaS 랜딩, 제품 데모
- 핵심 구문: `UI 텍스트와 창 제목이 실제로 읽히고, 반사광과 디스플레이 질감`

### DNA 4 · 업무 만화 · 에디토리얼 코믹
- 웹툰과 에디토리얼 중간 톤 + 컷 리듬
- 용도: 조직 교육, SaaS 사용 안내
- 핵심 구문: `실제 업무 대화처럼 짧고 또렷한 문장, 현실적인 속도감`

### DNA 5 · 한국형 럭셔리 호스피탈리티
- 자연광·건축재·리넨 + 넓은 여백
- 용도: 호텔, 라이프스타일 브랜드
- 핵심 구문: `감성적이지만 지나치게 관광 포스터 같지 않은 상업 사진, 기억에 남는 브랜딩 중심`

### DNA 6 · 미니멀 한지/동양 정서
- 여백 중심 + 한지/동양 텍스처
- 용도: 명절 캠페인, 프리미엄 시즌
- 핵심 구문: `한국적 겨울 정취를 담은 미니멀 한지 레이어 아트`
